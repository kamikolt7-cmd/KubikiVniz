# Tetris
«Тетрис» —Игрок управляет падающими фигурами различной формы («тетрамино»), пытаясь расположить их так, чтобы заполнить горизонтальные линии без промежутков. Заполненная линия исчезает, освобождая пространство. Цель — набрать максимальное количество очков, пока фигуры не заполнят всё игровое поле сверху.
